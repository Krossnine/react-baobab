## Getting Started

### Installation

```shell
$ git clone git@github.com:Krossnine/react-baobab.git
$ cd react-baobab
$ npm install
```

### Launch application

```shell
$ npm start
```

### Start application in dev mode

```shell
$ npm run watch
```
